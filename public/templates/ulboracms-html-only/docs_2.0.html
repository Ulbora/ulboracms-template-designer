Ver. 2.0 Documentation
Installing Ulbora CMS
Installing or updating Ulbora CMS is very easy. Just install the npm module Ulbora CMS Installer from command line as specified in the instructions.
Install a New Ulbora CMS App
Follow the instructions above and type: ulboracms-installer -h from command line for help.
Default Path and Version
The installer defaults to the latest version. The default path is shown below:
  • Linux:/var/www-node
  • Windows: c:\nodeapps
Install Ulbora CMS Dependencies
Navigate to the location of the new Ulbora CMS application and type: npm install
Starting Ulbora CMS
Navigate to the location of the server.js file in the new Ulbora CMS application and type: node server.js
Managing Ulbora CMS
  • To access the administration screens, type: yourURL:8080/admin
  • Enter username admin
  • Enter password admin
  • Click Users in the menu
  • Change the admin password to a strong password
